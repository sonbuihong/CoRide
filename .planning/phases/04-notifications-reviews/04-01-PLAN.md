<task type="auto">
  <name>Cập nhật Prisma Schema</name>
  <files>code/packages/database/prisma/schema.prisma</files>
  <action>
    - Thêm model `Notification` với các trường:
      - `id` (uuid)
      - `userId` (String, reference to User)
      - `title` (String)
      - `content` (String)
      - `type` (String, e.g., 'BOOKING_REQUEST', 'BOOKING_STATUS', 'RIDE_COMPLETED')
      - `isRead` (Boolean, default: false)
      - `createdAt` (DateTime, default: now)
    - Cập nhật model `User`:
      - Thêm `ratingCount` (Int, default: 0) để hỗ trợ tính toán rating nhanh hơn (Denormalization).
      - Thêm quan hệ `notifications Notification[]`.
    - Đảm bảo model `Review` đã có sẵn (đã kiểm tra ở bước Discovery).
    - Chạy lệnh: `npx turbo db:push` hoặc `npx prisma db push` trong thư mục database.
  </action>
  <verify>
    <automated>cd code/packages/database && npx prisma validate</automated>
  </verify>
  <done>Database schema được cập nhật thành công với model Notification và User.ratingCount</done>
</task>
