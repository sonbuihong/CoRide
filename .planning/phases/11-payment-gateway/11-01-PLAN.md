  <files>code/packages/database/prisma/schema.prisma</files>
  <action>
    Thêm các model và enum sau vào schema.prisma:
    1. Enum `TransactionStatus`: PENDING, SUCCESS, FAILED.

    7. Cập nhật model `Booking`: Thêm field `paymentStatus` (PaymentStatus, default UNPAID).

    - `cd code/packages/database && npx prisma migrate dev --name add_payment_system`
