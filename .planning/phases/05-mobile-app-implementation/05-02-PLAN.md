<objective>
Triển khai hệ thống Xác thực (Authentication) và quản lý Hồ sơ người dùng trên ứng dụng Mobile.
Mục tiêu:
1. Thiết lập lưu trữ token bảo mật với Expo SecureStore.
2. Xây dựng logic gọi API Login/Register và quản lý trạng thái auth.
3. Triển khai giao diện màn hình Đăng nhập, Đăng ký và Profile.
4. Bảo vệ các tuyến đường (Route Protection) dựa trên trạng thái đăng nhập.
</objective>

  <files>code/apps/mobile/src/services/auth-storage.ts, code/apps/mobile/src/services/auth.service.ts</files>
  <action>
    - Tạo `auth-storage.ts` sử dụng `expo-secure-store`.
    - Tạo `api.ts` (Axios client) cấu hình `baseURL` trỏ về Backend IP.
    - Thêm interceptor để tự động gắn Bearer token vào header nếu có.
    - Implement `authService` với các hàm `login`, `register`, `logout`.
  </action>
