<feature>
  <name>Hệ thống xác thực JWT & Middleware</name>
  <files>code/apps/backend/src/services/auth.service.ts, code/apps/backend/src/middlewares/auth.middleware.ts</files>
  <behavior>
    - Đăng ký: Nhận email/password -> Lưu user mới với password đã hash -> Trả về 201.
    - Đăng nhập: Kiểm tra email/password -> Trả về Access Token & Refresh Token (HttpOnly Cookie) -> Trả về 200.
    - Refresh Token: Nhận Refresh Token hợp lệ -> Hủy token cũ -> Trả về cặp token mới.
    - Auth Middleware: Kiểm tra Authorization header (Bearer) -> Verify JWT -> Gắn user vào req.user -> next().
    - Auth Middleware: Token không hợp lệ hoặc thiếu -> Trả về 401.
  </behavior>
  <implementation>
    - Sử dụng `bcrypt` để hash mật khẩu (salt round 10).
    - Sử dụng `jose` để ký và xác thực JWT.
    - Lưu Refresh Token vào Database để hỗ trợ rotation/revocation.
    - Express 5.x async middleware support.
  </implementation>
</feature>
